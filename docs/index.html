Private/internal electrical engineering standards assistant

Asisten Standar IEEE-IEC-SPLN

Prototype Streamlit untuk membaca PDF standar dari Google Drive, membuat indeks JSONL lokal, mencari cuplikan dengan TF-IDF, dan menjawab pertanyaan dengan Gemini hanya dari konteks terpilih.

Buka Repository Deploy di Streamlit Cloud

Retrieval-first

Mengambil cuplikan relevan terlebih dahulu sebelum memanggil Gemini.

Google Drive storage

PDF disimpan di Google Drive dan disinkronkan ke penyimpanan lokal sementara.

OCR fallback

Tesseract OCR tersedia untuk PDF scan saat membangun ulang indeks.

Fitur utama

  • Ekstraksi PDF dengan PyMuPDF dan OCR fallback.
  • Indeks ringan berbasis chunks.jsonl dan standards_index.json.
  • Pencarian lokal tanpa database eksternal.
  • Mode tanya standar dan perbandingan IEC vs IEEE vs SPLN.
  • Jawaban dengan sitasi badan standar, file/nomor standar, klausul, dan halaman.

Catatan privasi

Halaman ini tidak memuat PDF, teks standar, API key, atau kredensial Google Drive. Aplikasi utama harus dijalankan di Streamlit Cloud dengan secrets pribadi.

Cara menjalankan aplikasi

  • Deploy repository ini ke Streamlit Community Cloud.
  • Tambahkan GEMINI_API_KEY, GOOGLE_DRIVE_FOLDER_ID, dan GOOGLE_SERVICE_ACCOUNT_JSON pada Streamlit secrets.
  • Bagikan folder Google Drive PDF ke email service account.
  • Di aplikasi, klik Sync from Google Drive lalu Rebuild Index.
